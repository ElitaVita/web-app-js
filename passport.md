## 2. Описание предметной области

Система предназначена для автоматизации процесса регистрации посетителей на проходной организации. Она позволяет фиксировать ФИО посетителя, цель визита, время входа, а также ответственного сотрудника, к которому направляется посетитель. Система ведёт учёт времени выхода посетителя и формирует журнал посещений. Основные пользователи — сотрудники охраны на посту, администраторы системы и сами посетители (для предварительной записи). Система решает задачи контроля доступа, учёта рабочего времени для сторонних лиц и формирования отчётности о посещениях. В перспективе система может быть интегрирована с пропускной системой и СКУД.

| Администратор | Управляет списком ответственных сотрудников, просматривает журнал посещений, формирует отчёты, настраивает параметры системы |

## 5. Пользовательские сценарии (Use Case)

| № | Сценарий | Краткое описание |

### Лабораторная работа №2

**Основы синтаксиса JavaScript. Операторы языка**

| Реализуемый элемент         | Описание |
| --------------------------- | -------- |
| Переменные и типы данных | Создана константа `appConfig` с полями appTitle, defaultStatus, minValueForFilter. Создан массив `visitors` из 6 объектов с полями id, title, value, status, createdAt |

| Циклические конструкции | Через цикл `for` подсчитано количество объектов со статусом "new". Результат выведен в консоль |
